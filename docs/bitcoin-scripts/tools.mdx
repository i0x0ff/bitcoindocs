# Tools

List of various tools that can help you work with Bitcoin scripts.

- [btcdeb](https://github.com/bitcoin-core/btcdeb): This is a set of tools used to debug or construct scripts for use in Bitcoin
- [scriptwiz](https://ide.scriptwiz.app/): Online Script IDE for Bitcoin and Liquid network
