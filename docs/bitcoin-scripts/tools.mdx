# Tools

List of various tools that can help you work with Bitcoin scripts.

- [btcdeb](https://github.com/bitcoin-core/btcdeb): This is a set of tools used to debug or construct scripts
- [scriptwiz](https://ide.scriptwiz.app/): Online Script IDE for Bitcoin and Liquid network
- [tapsim](https://github.com/halseth/tapsim): Tool to debug, visualize VM state and construct Tapscript transactions
